Before you start

Use this tool to match geometric shapes on a map with blocks of text.

Your answers will be used to help link different categories of data in the Land Register.

What you need to do

You’ll be assigned one or more jobs.

Each job contains a list of tasks with a reference number, for example AB123456.

Each task contains:

  • a list of texts, for example text 1
  • a series of shapes in a map

You’ll answer yes or no for all the shapes whether they are included in each text.

When you’ve answered for all the shapes against a text, you can mark the text as done. You can go back and change your answers.

You’ll answer for the same shapes in any other texts in the task.

When all texts in a task are marked as done, you can submit the task. You can’t change your answers after you’ve submitted a task.

When all the tasks in a job are submitted, the job will be complete.

What we mean by matches, styles and shapes

Matches

A shape is included in the text if it includes the style properties. It may also have a label, such as an arrow and number or text.

A text may match more than one shape.

A description of the style is included above the map. This should help understand the style being shown but is not part of your matching decision.

You don’t need to consider other information in the text, such as legal agreements or the type of land or feature.

Styles

The style properties can include:

  • edging, which is the border of a shape
  • tinting, which is the colour of a shape
  • hatching, which is used as a texture for some shapes

Colours

Colours include:

  • green
  • red
  • blue
  • pink
  • brown
  • mauve
  • yellow

Shapes

Shapes include:

  • polygons, which usually cover an area of space 
  • lines, which can be solid or broken
