The other admin user need for managing workload is to reassign an active job, such as when an annotator is unable to complete a job and they need somone else to complete it.

If a job is in progress with some of the titles already submitted, the next annotator only needs to do the reamining titles that havne’t been submitted with a ‘to do’ or ‘in progress’ status.

If a job is ‘to do’, then the new annotator will be assigned the whole job.

If a job is complete, it can’t be reassigned. However, the admin can upload and assign a new job which contains some of the same titles.

The reassign page includes:

  • the job number in the h1
  • inset text to show the current annotator
  • radio buttons to select the new annotator

The user is then returned to the jobs status page with a success alert to show the job number and the new annotator. The assigned date and time is reset to the current timestamp.
